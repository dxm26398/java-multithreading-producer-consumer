# Java Multithreading — Producer Consumer

A multithreaded Java program demonstrating the Producer-Consumer pattern using thread synchronization.

## Features
- 1 Producer thread generating data items
- 3 Consumer threads processing items concurrently
- Thread-safe shared buffer using ReentrantLock and Condition
- Color-coded console output per thread
- Graceful thread termination handling

## Technologies
- Java
- Multithreading (Thread class)
- ReentrantLock and Condition (java.util.concurrent)
- LinkedList queue
